# Unit Converter Web App

## Project Description

This is a simple web-based unit converter built using Python and Flask.
It can convert units of Length, Weight, and Temperature.

## Features

* Convert Length units (meter, km, inch, etc.)
* Convert Weight units (kg, gram, pound, etc.)
* Convert Temperature (Celsius, Fahrenheit, Kelvin)

## Technologies Used

* Python
* Flask
* HTML

## How to Run

1. Install Flask:
   pip install flask

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
